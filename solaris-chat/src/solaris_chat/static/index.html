Soul

Solaris's durable identity (SOUL.md). One soul — personalities are per-session overlays on top of it. Edits apply live, no restart.

Model

Which model everyday chats run on (admin only). Schnell is the fast e2b model; Gründlich is the larger 12b model for harder questions. The household ("Zuhause") chat runs its own model — fast by default — set below. Applies to the next turn — no restart.

⚠️ A larger model on the household ("Zuhause") chat means slower voice/quick answers — the hot path is latency-sensitive. Keep it on the fast model unless you accept the latency.

Add a model

Pull a model into the local Ollama by tag or URL — a registry tag (gemma4:12b) or a Hugging Face GGUF repo (hf.co/<repo>:<quant>). Once pulled it appears in the pickers above. Admin only.

When Thinking is on (the Thinking selector in the chat header), Solaris reasons before answering and shows that reasoning in a separate block. Choose how it is shown. With Thinking off there is no reasoning block.
